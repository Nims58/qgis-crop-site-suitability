Installations
Crop Site Suitability Mapping Plugin can be downloaded from from official QGIS plugin repository or from https://github.com/Nims58/qgis-crop-site-suitability

Algorithm Description
This Plugin offers an easy to use, equal weighted overlay method to derive suitable sites for crop cultivation. It streamlines the process by integrating reclassification and threshold-based analysis into a single, user-friendly workflow.

Input Parameters
Input Layers: Climatic, topographic, and soil data
Input Parameters: Thresholds (min, max) as required to derive suitability class [Highly suitable, Moderately Suitable, Marginally Suitable and Not Suitable] area for a particular crop cultivation. 

Outputs
Result: Binary map showing suitable area.
